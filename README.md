# Music-service
Каталог музыки: 
- музыка: 
  - инвентарный номер, 
  - жанр, 
  - название, 
  - исполнитель, 
  - композитор,
  - тип носителя: 
    - CD/кассета/..., 
  - название фирмы звукозаписи, 
  - дата записи;

- пользователь: 
  - ФИО, 
  - адрес, 
  - телефон, 
  - логин, 
  - пароль; 

- плейлист.
